# Reverse Run

## ゲームの内容
Playerを動かし、コインを集めるゲーム
Easyはそのままの操作
Normalは上下or左右のどちらかが操作が反転
Hardは上下左右の操作が反転

![画面イメージ](docs/images/game_image01.png)

## 操作説明
WASDまたは方向キーで操作する
